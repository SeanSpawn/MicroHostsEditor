Micro Hosts Editor - это небольшая, простая в использовании, быстрая и удобная утилита для работы с файлом Hosts с открытым исходным кодом. При помощи лёгкого в освоении графического интерфейса, вы сможете управлять содержимым данного файла, не боясь что-ли повредить в системе.

Для полноценного функционирования программы необходимы права локального администратора. При их отсутствии утилита сможет работать лишь в режиме только для чтения. Чтобы запустить программу с правами администратора, нажмите правой кнопкой по ярлыку Micro Hosts Editor, и в контекстном меню выберите пункт Запуск от имени администратора.

Перед началом использования, пожалуйста, прочитайте официальную справочную систему (выберите пункт меню Справка - Справочная система, либо нажмите клавишу F1 на клавиатуре в главном окне программы).

Для работы данного приложения необходим установленный распространяемый пакет .NET Framework 4.8 или более новая версия. Загрузить его можно либо вручную с официального сайта, либо ответив утвердительно на запрос операционной системы при первом запуске.

Если вы хотите сообщить об ошибке в программе, либо предложить новую функцию, создайте новую заявку в нашей системе отслеживания ошибок (Справка - Сообщить об ошибке).
