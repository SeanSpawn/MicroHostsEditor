Micro Hosts Editor is a small, simple, cross-platform and completely free open-source Hosts editor. You can easily edit your Hosts file using simple GUI.

In order to make changes to Hosts file, this program need local administrator rights. Without them it will work only in read-only mode. Just press the right mouse button on Micro Hosts Editor’s shortcut and select Run as administrator.

Before using, please read the official help manual (select Help - Show help or press F1 key in the main window).

You must have .NET Framework 4.8 or newer version installed in order to run this application. You can download it from the official website or let the operating system install it automatically on first launch.

If you want to request a new feature or report a bug, feel free to open a new ticket in our official bug tracker (Help - Report a bug).
